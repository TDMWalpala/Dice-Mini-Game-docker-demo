# 🎲 Dice Mini Game 🐳 docker demo 🎲

## Docker and Docker Compose.

Build a Docker Image

```bash
  docker build -t demo:v1 .
```

Run a Docker Image

```bash
  docker run demo:v1
```

Run a Docker Image with port mapping:

```bash
  docker run -p 8080:80 demo:v1
```

Build using Docker Compose

```bash
  docker-compose up
```
